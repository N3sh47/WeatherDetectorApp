About

This is my project where I thought weather forecast can help people get knowlegde about weather. This web app can even help students when doing a weather study at school or any other field. This weather detector provides a range of weather related products like;

  • Country_Code and Temperature

    This API provides a technology that is able to tell country codes and also measure the hotness or coldness expressed in Kelvins.

  • Humidity and Pressure

    It provides a feature that outputs the actual Humudity and Pressure of every point globally.

  • Longitude and Latitude

    It provides a feature where it shows geographic coordinate that specifies the East–West and North-South position of a point on the surface of the Earth.
